#  Mock AI Symptom Tracker

A **mock AI symptom checker** web application built with **React**, **TypeScript**, **Tailwind CSS**, and **React Router**.  
This app allows users to input symptoms and get guidance based on predefined mock responses for educational or demo purposes.

---

## 📌 Features

- Input one or more symptoms separated by spaces (e.g., `fever headache`).
- Autocomplete suggestions as you type symptoms.
- Shows categorized recommendations with severity levels:
  - **Info** – General advice
  - **Warning** – Possible causes
  - **Danger** – Seek medical attention
- Clear input or go back easily.
- Mobile-responsive design using Tailwind CSS.

---

## 💻 Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS, Framer Motion
- **Routing:** React Router DOM
- **Data:** Mock symptom response data in TypeScript (`mockResponses`)
